# Document Ingestion & Search System (TF-IDF)

This project implements a document search engine that uses the **TF-IDF (Term Frequency-Inverse Document Frequency)** algorithm to rank documents based on a single-word query. The system is built with a Node.js backend and a React frontend, fully containerized using Docker.
